# Flodo Task Manager

## Tech Stack
- Flutter (Frontend)
- Flask (Backend)
- REST API

## Features
- Create, read, update, delete tasks
- Status filtering
- Search functionality

## How to Run

### Backend
cd backend  
pip install -r requirements.txt  
python run.py  

### Flutter App
cd flutter_app  
flutter pub get  
flutter run

## Testing
Basic API tests can be written using unittest to validate endpoints like GET /api/tasks and POST /api/tasks.
